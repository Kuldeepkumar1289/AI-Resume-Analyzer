# AI Resume Analyzer 🚀

An AI-powered Resume Analysis System built with Django that analyzes resumes and provides useful career insights.

## 📌 Features

* 📄 Resume Upload (PDF/DOCX)
* 🔍 Extract Resume Information
* 🧠 AI-based Skill Detection
* 📊 ATS Score Analysis
* 🎯 Career Role Prediction
* 🚀 Skill Gap Suggestions
* 🛣️ Personalized Learning Roadmap

## 🛠️ Tech Stack

* Python
* Django
* HTML, CSS, JavaScript
* SQLite/MySQL
* NLP & AI Concepts

## ⚙️ How to Run

```bash
pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```

## 📌 About Project

This project helps students and job seekers analyze their resumes, find missing skills, improve their profiles, and get career guidance using AI.
